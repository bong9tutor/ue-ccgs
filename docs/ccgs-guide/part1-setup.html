사전 준비 — 프로그램 설치하기

게임 개발을 시작하기 위해 필요한 모든 프로그램을 컴퓨터에 설치합니다.
처음 하시는 분도 그대로 따라하시면 됩니다.

1

Git 설치하기

Git이 뭔가요?

Git은 파일의 변경 이력을 관리하는 프로그램입니다. 게임을 만들면서 "어제 버전으로 돌아가고 싶다"할 때 사용합니다. CCGS 프로젝트를 다운로드할 때도 필요합니다.

Windows에서 설치하기

  • ✓웹 브라우저에서 git-scm.com/download/win 접속
  • ✓"Click here to download" 클릭하여 설치 파일 다운로드
  • ✓다운로드된 파일 실행 → 모든 옵션을 기본값 그대로 "Next" 클릭
  • ✓마지막에 "Install" 클릭 → 설치 완료

설치 확인하기

시작 메뉴에서 "명령 프롬프트" 또는 "터미널"을 검색하여 실행하세요. (Mac은 "터미널" 앱)

terminal
$ git --version git version 2.47.1 ← 이런 식으로 버전 번호가 나오면 성공!
⚠
"git을 찾을 수 없습니다" 오류가 나오면?

컴퓨터를 재시작한 후 다시 시도하세요. 그래도 안 되면 Git을 다시 설치하되, 설치 중 "Add to PATH" 옵션이 체크되어 있는지 확인하세요.

2

Node.js 설치하기

Node.js가 뭔가요?

Claude Code를 실행하기 위해 필요한 프로그램입니다. Claude Code가 Node.js 위에서 동작합니다.

설치 순서

  • ✓nodejs.org 접속
  • ✓초록색 "LTS" 버튼 클릭하여 다운로드 (Recommended 버전)
  • ✓다운로드된 파일 실행 → 기본값 그대로 Next → Install
설치 확인
$ node --version v22.x.x ← 버전 번호가 나오면 성공! $ npm --version 10.x.x ← npm도 같이 설치됩니다
3

Claude Code 설치하기

Claude Code가 뭔가요?

Anthropic이 만든 AI 코딩 도구입니다. 터미널에서 Claude AI와 대화하며 코드를 작성할 수 있습니다. CCGS는 이 Claude Code 위에서 동작하는 프레임워크입니다.

설치하기

터미널에 아래 명령어를 그대로 복사해서 붙여넣고 Enter를 누르세요.

terminal
$ npm install -g @anthropic-ai/claude-code ... 설치 진행 중 (1~2분 소요) ... added 1 package in 30s

Anthropic API 키 설정 (필수)

Claude Code를 사용하려면 Anthropic 계정과 API 키가 필요합니다.

  • ✓console.anthropic.com 접속 → 회원가입 또는 로그인
  • ✓좌측 메뉴 "API Keys" 클릭
  • ✓"Create Key" 클릭 → 이름 입력 (예: "game-dev") → 키 복사
  • ✓터미널에서 아래 명령어 실행 (키를 따옴표 안에 붙여넣기):
Windows
# Windows (명령 프롬프트) $ set ANTHROPIC_API_KEY=sk-ant-여기에-복사한-키-붙여넣기 # Mac / Linux (터미널) $ export ANTHROPIC_API_KEY=sk-ant-여기에-복사한-키-붙여넣기
✅
매번 입력하기 귀찮다면?

Claude Code를 처음 실행하면 자동으로 로그인/인증 과정을 안내합니다. 화면에 나오는 안내를 따라가시면 됩니다.

설치 확인

terminal
$ claude --version claude-code x.x.x ← 버전 번호가 나오면 성공!

항목최소 사양권장 사양

설치 순서

  • ✓Epic Games 계정 생성 (무료) → 로그인

5

가이드 & 샘플 프로젝트 다운로드 (클론)

"클론"이 뭔가요?

GitHub에 있는 프로젝트를 내 컴퓨터로 복사하는 것입니다. "다운로드"와 비슷하지만 Git의 버전 관리 기능이 함께 옵니다.

무엇을 받게 되나요?

아래 명령어로 다운로드하는 ue-ccgs 저장소에는 3가지가 모두 포함되어 있어서 바로 실습을 시작할 수 있습니다.

구성 요소위치역할
📖 가이드 사이트docs/ccgs-guide/지금 보고 있는 5개 파트 HTML
🧠 CCGS 프레임워크.claude/ · design/ · production/에이전트·스킬·훅이 즉시 동작
🎮 샘플 UE 프로젝트MadeByClaudeCode/UE 5.6 Blank C++ 스켈레톤

실행하기

터미널에서 아래 명령어를 한 줄씩 입력하고 Enter를 누르세요.

terminal
# 1. 원하는 작업 폴더로 이동 (예: 바탕화면) $ cd Desktop # 2. 통합 가이드 + CCGS + 샘플 UE 프로젝트 다운로드 $ git clone https://github.com/bong9tutor/ue-ccgs.git Cloning into 'ue-ccgs'... remote: Enumerating objects: ..., done. ... (다운로드 중) ... # 3. 프로젝트 폴더로 이동 $ cd ue-ccgs # 4. 폴더 구조 확인 $ ls .claude/ MadeByClaudeCode/ design/ docs/ production/ CLAUDE.md README.md
💡
자기만의 게임을 만들고 싶다면?

이 저장소를 GitHub에서 Fork하거나, 클론 후 MadeByClaudeCode/ 폴더와 .uproject 파일 이름을 원하는 게임 이름으로 바꿔 사용하세요.

6

샘플 UE 프로젝트 열어보기

MadeByClaudeCode 프로젝트란?

MadeByClaudeCode/ 는 가이드 실습을 위한 UE 5.6 Blank C++ 스켈레톤 프로젝트입니다. CCGS가 /dev-story 명령어로 코드를 구현할 때 이 프로젝트의 Source/ 폴더에 C++ 파일을 생성합니다.

프로젝트 열기

  • ✓파일 탐색기에서 ue-ccgs/MadeByClaudeCode/ 폴더로 이동
  • ✓MadeByClaudeCode.uproject 파일을 더블클릭
  • ✓"Missing Modules" 창이 뜨면 "Yes" 클릭 (C++ 모듈 빌드)
  • ✓빌드 완료 후 UE 5.6 에디터가 실행됩니다
✅
첫 빌드는 시간이 걸립니다.

5~20분 정도 소요될 수 있습니다. 완료 후에는 편집기 테마, 레벨 로드 등이 자동으로 진행됩니다. 빌드 중에는 에디터를 닫지 마세요.

💡
에디터는 지금 바로 닫아도 됩니다.

CCGS가 코드를 생성한 후 UE 에디터로 돌아와 컴파일·테스트를 진행하면 됩니다. Part 2~5 진행 중에는 에디터를 닫아둬도 무방합니다.

7

첫 Claude Code 세션 시작하기

Claude 세션 시작

프로젝트 루트 폴더(ue-ccgs) 안에서 아래 명령어를 입력합니다. MadeByClaudeCode/ 안이 아니라 한 단계 위인 ue-ccgs/에서 실행해야 합니다 — CCGS가 .claude/ 폴더를 찾기 때문입니다.

terminal — 첫 세션
# 현재 위치 확인 (ue-ccgs 루트여야 함) $ pwd /.../Desktop/ue-ccgs # Claude Code 실행 $ claude Claude Code v2.1.x Opus 4.x · C:\...\ue-ccgs > _ # 화면 하단에 아래와 같은 상태줄이 보이면 CCGS 정상 동작! ctx: -- | Opus 4.x | Concept
🎉
CCGS 동작 확인 포인트

Claude Code 실행 후 화면 하단 상태줄에 Concept, Design, Sprint 📝 같은 단어가 보이면 CCGS의 .claude/statusline.sh 가 정상 작동하는 것입니다. 아직 게임을 만들기 전이므로 기본값인 Concept 이 표시됩니다.

💡
세션 시작 배너는 보이지 않습니다

CCGS의 session-start.sh 훅은 브랜치·최근 커밋·스프린트 정보를 수집하지만, Claude Code v2.1.x 에서는 이 내용이 사용자 터미널이 아니라 Claude(AI)의 컨텍스트에만 전달됩니다. 즉, Claude는 프로젝트 상태를 알고 있지만 화면에는 표시되지 않습니다. 하단 상태줄로 축약 정보가 대신 표시됩니다.

실제 동작 검증: /start 입력해보기

Claude 프롬프트(> _)에 아래 명령어를 입력해 보세요.

동작 테스트
나 > /start Claude: "Claude Code Game Studios에 오신 것을 환영합니다! 지금 게임 아이디어 관점에서 어디쯤 계신가요? A) 아이디어가 전혀 없어요 B) 모호한 아이디어가 있어요 ..."

위와 같이 CCGS Orchestrator 에이전트의 응답이 나오면 설치·설정이 모두 완료된 것입니다. 구체적인 사용법은 Part 2에서 이어집니다.

⚠
상태줄이 안 보이거나 /start가 인식 안 된다면?

다음을 확인하세요: (1) ue-ccgs 폴더 안에서 claude를 실행했는지 (MadeByClaudeCode/ 안이 아님!) (2) .claude 폴더가 존재하는지 (ls -la로 확인) (3) API 키가 설정되어 있는지 (4) Claude Code 버전이 v2.1 이상인지 (claude --version)
