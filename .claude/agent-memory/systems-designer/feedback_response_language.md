모든 응답, 분석, 권고, 질문은 한국어로 작성한다.

**Why:** 이 저장소의 CLAUDE.md가 "사용자와의 대화 및 문서 작성의 기본 언어는 한국어"로 명시. 사용자 auto-memory도 동일 규칙 확인.

**How to apply:** 영어 기술 용어(CardId, FOnCardOffered 등)는 그대로 사용하되, 설명 문장과 권고 문장은 한국어로.
