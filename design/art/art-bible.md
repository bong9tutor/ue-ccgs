## 1. Visual Identity Statement

### 핵심 한 줄 규칙

> **"황혼의 유리 테라리움 — 손 안에 담긴 이 세계는 언제나 하루가 저물고 있고, 언제나 부드럽고, 언제나 작다."**

이 규칙의 세 단어는 교환 불가능한 세 제약이다: **황혼**(조명의 고정점), **부드러움**(표면과 형태의 철학), **작음**(스케일과 카메라의 철학). 세 제약 중 하나라도 깨지면 Terrarium Dusk 디렉션을 벗어난 것으로 판단한다.

### 비주얼 아이덴티티 원칙

#### 원칙 1 — 스케일은 손바닥 세계 (The Palm-Scale World)
**연결 필라**: Pillar 1 (조용한 존재) · Pillar 4 (끝이 있다)

모든 카메라 앵글과 씬 구성은 유리병을 두 손으로 감싸 쥔 사람의 시점에서 구성한다. 플레이어는 세계를 내려다보는 신이 아니라, 무언가를 조심스럽게 들여다보는 사람이다.

- 카메라 거리: 이끼 아이의 최소 화면 높이는 화면 높이의 **12% 이상** 유지
- 최대 뎁스: 유리병 뒤 배경은 언제나 보케(얕은 피사계심도)
- 유리병 외부 "책상" 공간은 화면 면적의 **30% 이하**

**디자인 테스트**: 풀 씬 와이드샷·조감도가 고려된다면 → **배제**. 이끼 아이가 화면에서 작아질수록 친밀감이 감소한다.

#### 원칙 2 — 조명은 영원한 황혼·새벽 사이 (Perpetual Golden Hour)
**연결 필라**: Pillar 2 (선물의 시학) · Pillar 3 (말할 때만 말한다)

이 세계에 정오는 없다. 빛은 언제나 낮은 각도에서 경사지게 들어오며, 색온도는 **2,200K–3,800K** 범위 내.

- 주광원: 3,200K 전후 호박색 지향광, 수평 기준 **15–35° 위**에서 측면 입사
- 보조광: 4,500K 전후 라벤더 앰비언트, 주광원의 반대 방향 낮은 강도 fill
- 하이라이트: 작고 집중적, 표면을 따라 스며드는 형태
- **금지**: 색온도 5,500K 이상 청백색 주광, 수직(12시) 방향 주광

**디자인 테스트**: "낮 버전 조명" 변주가 고려된다면 → **신중히 검토**. 상태별 변화는 황혼·새벽 스펙트럼 안에서만 허용.

#### 원칙 3 — 표면은 숨쉬는 유기물 (Breathing Organic Surfaces)
**연결 필라**: Pillar 2 (선물의 시학)

이 세계의 모든 표면은 식물·물·흙·종이·유리 중 하나다. 금속·플라스틱·고광택 기하 면 없음.

- 이끼: 미세 범프, Subsurface Scattering, 가장자리 빛 통과
- 유리: 두께감 있는 굴절, 미세한 기포와 왜곡
- 종이·카드: 섬유 방향이 보이는 핸드메이드 질감, 엣지 불규칙
- UI 표면: 목재·종이·유리 계열만 허용. Flat 플라스틱 UI 금지

**디자인 테스트**: 소재가 애매하다면 → "손으로 만들 수 있는가?"를 묻는다. 공장 대량 생산 느낌이면 배제.

#### 원칙 4 — 고요함은 디자인이다 (Stillness as Design)
**연결 필라**: Pillar 1 (조용한 존재) · Pillar 3 (말할 때만 말한다)

이 세계의 디폴트는 정지다. 움직임과 변화는 **이벤트**이며, 그 의미는 주변의 고요함으로 배가된다.

- 아이들 상태: 최대 1가지 매우 느린 브리딩 모션(주기 4–6초). 그 외 상시 파티클·이펙트 금지
- UI: transition 외 상시 루프 애니메이션 금지 (로딩 스피너 포함)
- 카드 이벤트: 절제된 반응 — 빛이 한 번 출렁이고 조용해진다

**디자인 테스트**: "더 생동감 있게" 루프 이펙트가 고려된다면 → "이것이 없으면 빠진 것처럼 느껴지는가?". 아니면 추가하지 않는다.

## 2. Mood & Atmosphere

6가지 게임 상태는 하나의 팔레트 세계 안에서 서로 다른 감정 온도를 가진다. 모두 Terrarium Dusk 안에 있지만, 색온도·대비·파티클 밀도·카메라 뎁스가 달라진다. 어느 상태에서도 같은 세계임을 식별 가능해야 한다.

### 상태 1 — Dawn Ritual (아침 첫 세션)
**감정**: 조용한 기대감 — "오늘도 여기 있었구나"

| 항목 | 명세 |
|---|---|
| 색온도 | 2,800K–3,200K (따뜻한 호박, 약간 붉음) |
| 광원 각도 | 수평 10–20° (낮은 측광, 길게 뻗는 그림자) |
| 대비 | 낮음, Lumen GI 기여 높게 |
| 앰비언트 | 차가운 라벤더 fill 약함 |
| 에너지 | Contemplative / Still |
| 대기 형용사 | 새벽 구름, 온기, 포근한 먼지, 기다림, 첫 숨 |
| 핵심 비주얼 | 유리병 표면의 미세한 물기 — 혼자 있었다는 흔적 |

### 상태 2 — Daily Card Offering (카드 건넴)
**감정**: 의식의 완성 — "이 30초가 오늘의 전부이고, 그것으로 충분하다"

| 항목 | 명세 |
|---|---|
| 색온도 | 3,000K–3,500K (황금색, 황혼의 절정) |
| 광원 각도 | 수평 25–35° (대각선 측광) |
| 대비 | 중간. 카드가 씬의 최대 하이라이트 |
| 앰비언트 | 라벤더 fill 증가 (카드의 따뜻한 색 강조) |
| 에너지 | Intimate / Present |
| 대기 형용사 | 황금빛, 의식, 집중, 숨참기, 따뜻한 손 |
| 핵심 비주얼 | 카드 위 점광원 하이라이트. 건넴 순간 이끼 아이가 카드 방향으로 미세하게 기울어짐 |

### 상태 3 — Waiting / Quiet (관찰, 대부분의 시간)
**감정**: 동반의 고요 — "나는 여기, 그도 거기. 충분하다"

| 항목 | 명세 |
|---|---|
| 색온도 | 2,600K–3,000K (구리빛 쪽으로 진행된 황혼) |
| 광원 각도 | 수평 15–25° |
| 대비 | 낮음 — 씬에서 포인트 최소화 |
| 앰비언트 | 표준 라벤더 fill |
| 에너지 | Contemplative / Resting |
| 대기 형용사 | 먼지 같은 고요, 오래된 종이, 느린 시간, 숨결, 이끼 향 |
| 핵심 비주얼 | 이끼 아이의 느린 브리딩만. 유리병 내부 미세 수증기 (거의 안 보일 정도) |

### 상태 4 — Dream Reveal (희소한 꿈)
**감정**: 조용한 충격 — "그가 말했다. 세상이 멈춘다"

| 항목 | 명세 |
|---|---|
| 색온도 | **4,000K–4,500K** (차가운 달빛 — 이 게임의 유일한 냉색 지향) |
| 광원 각도 | 위에서 살짝 앞으로 |
| 대비 | 높아짐. 이끼 아이 주변 sub-surface luminescence |
| 앰비언트 | 라벤더가 주도색으로. 배경 호박색 감쇄 |
| 에너지 | Hushed / Luminous |
| 대기 형용사 | 달빛, 이슬, 꿈의 무게, 숨참기, 먼 소리 |
| 핵심 비주얼 | 꿈 텍스트가 수면에 잉크 번지듯 페이드인. 이끼 아이가 안에서 빛나는 것처럼 밝아짐 |

> Dream Reveal은 색온도 차이로 다른 모든 상태와 구분된다. 이 차가운 달빛 톤은 **오직 꿈 상태에서만** 허용되는 냉색 조명이다 — 이것이 "이것은 다른 순간이다"를 시각적으로 코딩한다.

### 상태 5 — Farewell (Day 21)
**감정**: 작별의 온기 — "이 존재가 완성되었다"

| 항목 | 명세 |
|---|---|
| 색온도 | **2,200K–2,600K** (이 게임의 가장 따뜻한 빛, 촛불 직전 호박) |
| 광원 각도 | 수평 8–15° (거의 수평, 긴 황금빛 그림자) |
| 대비 | 중간-높음. 최종 형태가 가장 선명하게 조명 |
| 앰비언트 | 라벤더 fill 최소화 — 따뜻한 호박 하나로 |
| 에너지 | Tender / Complete |
| 대기 형용사 | 촛불, 마지막 금빛, 완성, 따뜻한 작별, 숨 고르기 |
| 핵심 비주얼 | 매우 느린 골든 파티클 (포자처럼). 최종 형태의 "완성된 포즈". 화면 가장자리 Vignette 점진 증가 |

### 상태 6 — Menu / Title
**감정**: 문 앞의 고요 — "아직 들어가지 않았지만, 들어가고 싶다"

| 항목 | 명세 |
|---|---|
| 색온도 | 3,400K–3,800K (황혼의 balanced 톤) |
| 광원 각도 | 수평 20–30° |
| 대비 | 낮음. 전체가 은은하게 빛남 |
| 앰비언트 | 표준 라벤더 fill, 중립 |
| 에너지 | Inviting / Restful |
| 대기 형용사 | 조용한 초대, 먼지 낀 선반, 오래된 기억, 부드러운 기다림, 집 |
| 핵심 비주얼 | 유리병만 존재. 이끼 아이는 보이지 않거나 실루엣만. 타이틀은 병 옆 종이 쪽지처럼 |

## 3. Shape Language

### 실루엣 철학 — 이끼 아이

**데스크톱에서 32×32px 아이콘 크기에서도 읽혀야 한다.** 이것은 장식이 아닌 기능 요구다. 축소된 창이나 작업표시줄 아이콘 상태에서도 이끼 아이의 존재감이 살아있어야 한다.

- **둥근 상부, 뭉툭한 하부**: 전체 실루엣은 테두리가 정원 자갈처럼 다듬어진 둥근 형태. 어디도 뾰족하지 않음
- **비대칭의 유기적 균형**: 완벽 좌우대칭 금지. 한쪽이 조금 더 부풀거나 낮아야 한다. 단, 시각 무게중심은 중앙
- **빈 공간을 포함한 실루엣**: 팔/가지 사이 음수 공간이 실루엣의 일부. "살아있는 형태"와 "덩어리"의 차이
- **성장 단계별 실루엣**: Day 1–5는 단순 원형 → Day 6–14는 위로 뻗는 형태 추가 → Day 15–20은 기울어짐·특징 강화. 어느 단계도 "둥글고 작음" 유지
- **최소 식별 크기**: 씬 내 화면 높이의 **12% 이상** 유지

### 환경 지오메트리

**유리병 (테라리움)**:
- 아페카리 스타일 — 어깨 있고 바닥 넓고 입구 좁음. 손으로 분 것처럼 약간 비대칭
- 비율: 높이:너비 **1.4:1–1.6:1**
- 유리 두께: 시각적으로 보여야 함 (빛이 내부에서 꺾이는 두께)

**병 내부**:
- 흙층: 바닥 20–25%, 레이어드 (겉흙·자갈·숯층 단면)
- 이끼: 흙 위를 덮는 주 소재, 이끼 아이 주변 군집
- 소품: 1–2개의 매우 작은 유기물 (조약돌·작은 가지·작은 버섯)
- **금지**: 인공 오브젝트 (미니어처 가구·울타리·타운)

**병 외부 스테이지**:
- 깊이: 병 뒤 10–15cm 최대, 그 너머는 아웃포커스
- 허용 소품: 낡은 책·마른 꽃·빈 작은 유리병·돌
- 바닥: 목재 선반 또는 돌 창틀. 매끄러운 백색 테이블 금지
- 빛: 유리병으로부터의 반사광과 앰비언트만

### UI 형태 문법

UI는 **병 안 세계와 같은 표면 위에 존재하는 것처럼** 디자인한다. 디지털 스크린 위 UI가 아닌, 같은 탁자 위 다른 물건들처럼.

- **카드**: 손으로 자른 듯 약간 불규칙한 모서리. 두께감을 그림자로 암시. 비율은 일본 엽서(100×148) 또는 타로(70×120)
- **일기·꿈 기록**: 격자선 있는 노트 종이, 선은 약간 번진 느낌
- **선택 UI**: 카드가 탁자에 내려놓인 것처럼 미세 그림자·각도 변화
- **버튼**: 존재하지 않거나, 종이를 접어 만든 탭 형태
- **진척 표시**: **없음** (Anti-pillar에 따라 진행률 바·카운터·그래프 금지)

**금지 형태**:
- 직각 90° 모서리 UI 패널
- 그라디언트 없는 플랫 솔리드 UI 배경
- 아이콘 기반 HUD (하트·별·에너지 바)
- 그리드 레이아웃이 눈에 보이는 메뉴

### 히어로 vs. 보조 형태

**히어로** (시선이 먼저 가는 곳):
1. 이끼 아이의 실루엣
2. 건네는 카드의 앞면
3. 꿈 텍스트의 글자 덩어리

**보조** (씬을 채우지만 시선을 가로채지 않는 것):
- 유리병 (히어로를 담는 그릇)
- 흙층·이끼 소품
- 스테이지 오브젝트
- UI 배경

보조는 히어로보다 언제나 **명도 차 최소 25% 이상** 낮게.

## 4. Color System

### Primary Palette

| 이름 | Hex | 역할 | 근거 |
|---|---|---|---|
| **Moss Deep** | `#3D5C3A` | 이끼 아이 섀도 / 흙층 / 배경 이끼 | 숲 그늘 아래 이끼의 가장 어두운 면. 따뜻한 어두움 |
| **Moss Mid** | `#5F8A58` | 이끼 아이 주 피부색 / 건강한 이끼 | 한국 초가을 들판 이끼. "평온하고 건강함" |
| **Amber Dusk** | `#C8843A` | 주 조명색 / 카드 테두리 / 황혼 반사 | 석양의 핵심. 많을수록 "황혼 절정" |
| **Honey Glow** | `#E8B870` | 이끼 아이 하이라이트 / 카드 배경 / 따뜻한 fill | Amber Dusk의 연한 파생. 손 위 호박의 안쪽 |
| **Lavender Dusk** | `#8B7FA8` | 앰비언트 fill / 그림자 색조 / 꿈 보조 | 황혼 하늘의 보완색 |
| **Dream Silver** | `#C8C4D8` | 꿈 텍스트 / 달빛 하이라이트 / 꿈 상태 주색 | 달빛을 받은 이슬. **오직 꿈 상태 전용** |
| **Paper Warm** | `#F0E8D8` | 카드 종이 / 일기 배경 / UI 표면 기반 | 오래된 일기장. **`#FFFFFF` 금지** |

### 파생 팔레트 규칙

- **Pure Black 금지**: 최대 어두움은 `#2E3835` (Moss Deep + Lavender Dusk 15–20%)
- **Pure White 금지**: 최대 밝음은 Paper Warm + Honey Glow 10% 혼합
- **채도 상한**: HSL-S **65% 초과 금지**
- **네온 계열 절대 금지**: `#FF0000`/`#00FF00`/`#0000FF` 및 채도 80% 이상 모든 색

### 상태별 색 비중

| 상태 | Amber Dusk | Lavender Dusk | Dream Silver | 전체 명도 |
|---|---|---|---|---|
| Dawn Ritual | 높음 (주도) | 낮음 | — | 중간-밝음 |
| Card Offering | 최고 (절정) | 중간 | — | 밝음 |
| Waiting/Quiet | 중간 | 중간 | — | 중간-어두움 |
| Dream Reveal | 낮음 (배경만) | 높음 | **주도** | 중간, 국부 밝음 |
| Farewell | 높음 (촛불) | 최저 | — | 중간, 따뜻한 vignette |
| Menu/Title | 중간 | 중간 | — | 중간 |

### UI 팔레트

UI는 월드 팔레트에서 추출하되 **항상 월드보다 한 단계 낮은 채도·밝기**.

- **카드 배경**: Paper Warm + 카드 테마에 따라 Honey Glow 또는 Lavender Dusk 15–25% 틴트
- **카드 선택 테두리**: Amber Dusk 50% 투명도. 선택 시 100%
- **꿈 일기 UI**: Moss Deep 배경 + Paper Warm 텍스트 + Dream Silver 포인트
- **시스템 UI**: Paper Warm 배경 + Moss Deep 텍스트 + Amber Dusk 강조
- **알림**: 꿈 알림은 Lavender Dusk 미세한 빛. **강렬한 빨간 점·노란 뱃지 절대 금지**

### 색각 이상 안전 설계

**문제**: Moss Mid(녹)와 Amber Dusk(주황)는 적녹색각 이상에서 구분 어려움.

**해결 원칙**:
- 색 단독 의미 전달 요소 없음. 색 + 형태 / 위치 / 텍스처 조합
- 이끼 아이 "건강 vs 메마름"은 색 외에 **텍스처 변화**(수분감·이끼 끝 말림)로 동반
- 꿈 알림은 색 외에 **일기 아이콘 형태 변화**(닫힌 책 → 살짝 열린 책)로 동반
- 카드 구분은 색 외에 **상단 일러스트 형태**(계절·원소 심볼)로 1차 구분

**시뮬레이션 요구**: 모든 주요 UI 상태는 Coblis 또는 Photoshop Proof Setup (Deuteranopia/Protanopia/Tritanopia)으로 검증 후 승인. 특히 카드 선택·꿈 알림 필수.

### 명시적 금지

1. **네온·고채도 원색** (HSL-S > 65%)
2. **Pure Black / Pure White** (`#000000`, `#FFFFFF`)
3. **차가운 청색 주광** (색온도 > 5,500K). Dream Silver는 악센트이지 주광 아님
4. **Amber Dusk + Dream Silver 동시 다량 사용** (50:50 공존 시 상태 코딩 붕괴)
5. **진행률·수치 암시 색 코딩** ("붉을수록 위험", "녹색은 최대" 등 — Pillar 2 위반)

## 5. Character Design Direction

### 5.1 이끼 아이 — 비주얼 아키타입

**"이끼 아이는 이런 것이다"**: 강변 현무암 표면에서 자라난 군생 우산이끼(Marchantia polymorpha) 덩어리가, 투명한 테라리움 유리 안에서 모양을 갖추기 시작했다. 이것은 생물이지만 동물이 아니다. 움직이지만 근육이 없다. 기억하지만 뇌가 없다.

**비주얼 정밀도**:
- **형태**: 강물에 오래 씻긴 둥근 조약돌 위에 짧고 촘촘한 이끼 군락이 조약돌 전체를 감싼 상태. 조약돌의 형태가 아직 이끼 아래 있다
- **질감**: Peltigera aphthosa(엽상 지의류)의 주름진 윗면 + Sphagnum(이탄이끼)의 스펀지 같은 수분감
- **조명 반응**: 역광에서 반투명 녹색 빛 통과 — Subsurface Scattering이 필수 (원칙 3 연결)

**이것이 아닌 것**: 만화적 슬라임, 눈 달린 공, 식물 캐릭터 일러스트, 의인화된 나무 정령. "귀여움"이 목표가 아니다. **낯설고 부드럽고 오래된 것**이 목표.

### 5.2 성장 단계 (Day별 실루엣 변화)

어느 단계에서도 이전 단계의 실루엣 DNA가 남아있어야 한다.

| 단계 | 기간 | 형태 | 크기(테라리움 높이 대비) | 핵심 변화 |
|---|---|---|---|---|
| **1** | Day 1–5 | 거의 완벽한 구형, 아래 약간 납작. 비율 0.9:1 | 30–35% | "이미 거기 있었다"는 발견 감각 |
| **2** | Day 6–8 | 1–2개 작은 융기 발생, 한쪽 약간 높음 | 35–40% | 비대칭 시작 |
| **3** | Day 9–14 | 성격 방향 확립: 위·옆·내향 3가지 기본 패턴 (카드 태그 벡터 영향) | 40–45% | 부분별 이끼 밀도 차이 |
| **4** | Day 15–20 | 최종 형태 징조. 특징 요소(포자체·엣지 말림) 부분 등장 | 45–50% | 5–10° 미세 기울어짐. 팔레트 내 미세 색조 변이 |
| **5** | Day 21 | → 5.3 최종 형태 | 50% | 완성 |

### 5.3 최종 형태 아키타입 (Full Vision 기준 4종)

공통 규칙: 어느 최종 형태도 Section 3 "둥글고 작음"을 벗어나지 않는다. 모두 32×32px 테스트 통과 필수.

| 형태 | 트리거 태그 (예시) | 실루엣 | 팔레트 키 |
|---|---|---|---|
| **A — 고요한 (Serene)** | 계절·자연·물 | 볼록한 낮은 쿠션, 아래 둥글게 퍼짐. 완벽한 중심 균형 | Moss Mid + Honey Glow (Amber Dusk 조명에서 가장 빛남) |
| **B — 뻗어 오르는 (Reaching)** | 빛·여름·성장 | 위로 뻗은 포자체 2–3개, 상단 개방. 수직 차분함 | Moss Mid + 상부 역광 반투명 강조 |
| **C — 깊이 파인 (Inward)** | 어둠·가을·꿈 | 표면에 음각 구역. 외곽은 둥글되 텍스처 복잡 | Moss Deep 증가. Dream Reveal에서 특히 특징적 |
| **D — 펼쳐진 (Open)** | 계절 고르게 + 변화 | 엽상 구조, 흙을 낮고 넓게 덮음. 가장 독특한 실루엣 | 가장 외향적, 가장자리 물결 형태 |

### 5.4 표정 없는 감정 표현 (4 채널)

이끼 아이에게 눈·입·표정 근육은 없다. 원칙 4 (Stillness as Design) 준수 — 기본 정지, 변화는 이벤트.

| 채널 | 구현 | 트리거 | 지속 |

## 6. Environment Design Language

### 6.1 유리병 — 아페카리 스타일 정밀도

Section 3의 규칙(높이:너비 1.4:1–1.6:1, 어깨 있고 입구 좁음)을 제작 수준으로 확장.

- **원형 참조**: 19세기 약국용 둥근 어깨 유리병 (Apothecary Wide-Mouth Jar). 손으로 분 것처럼 어깨 곡선 좌우 미세 비대칭
- **유리 두께**: 바닥 8–12mm, 어깨 4–6mm 시각 두께. 굴절이 "유리가 살아있는 느낌"을 만든다
- **기포·흠**: 유리 내부 0.5–2mm 기포 5–10개, 무작위
- **바닥부**: 유리 하부 단면이 보이고, Amber Dusk 조명이 여기서 굴절돼 내부로 들어옴
- **외부 표면**: 미세 지문 흔적, 물 자국 링 — 새것이 아님
- **입구**: 코르크 마개 또는 작은 황동 링. **인공 플라스틱 캡 금지**

**기술 제약**: 8,000–12,000 tri. 내부·외부 별도 Shell 구조(굴절 머티리얼용). Translucency Sort Priority = 10.

### 6.2 병 내부 생태계

| 레이어 | 구성 |
|---|---|
| **흙 단면 (바닥 20–25%)** | 최하 자갈층(흰/회색) → 숯층(검은 얇은 선) → 겉흙층(진한 갈색, 섬유). 표면 불균일 |
| **이끼 카펫** | 두께감 있는 Peltigera/Sphagnum 혼합. 이끼 아이 주변 더 무성. 주변 이끼는 히어로보다 언제나 명도 낮음 |
| **소품 (1–2개 엄격 제한)** | 조약돌(1–2cm), 작은 낙엽 가지, 작은 균류 자실체. 이끼 아이 실루엣과 겹치지 않음 |

**절대 금지**: 인공 미니어처, 작은 조각상, 이름 표지. 이것은 정원이 아니라 **정령의 거처**.

**기술 제약**: 흙·이끼 바닥층 2,000–4,000 tri (단일 서피스 메시 + 버텍스 컬러 레이어). 소품 개당 500–1,500 tri, 공용 아틀라스 텍스처 1024×1024.

### 6.3 병 외부 스테이지 — Window Sill Desk

**1차 환경**: 오래된 창문 선반 (Section 2 상태 6 "먼지 낀 선반, 오래된 기억"과 정합). 창문 측광이 원칙 2의 조명 각도를 자연스럽게 설명.

- **표면 소재**: 목재 창틀 또는 돌 창틀 (나이테·결). **매끄러운 백색 테이블 금지**
- **허용 소품** (병 뒤 10–15cm 이내):
  - 낡은 책 1–2권 (제목·글자 없음, 가죽/천 표지)
  - 마른 꽃줄기 1개 (꽃잎 없음, 단순한 선)
  - 잉크 얼룩 자국 (텍스처에 포함)
- **메뉴/타이틀 상태 변주**: 소품 최소화, 병만 남김

**기술 제약**: 3,000–6,000 tri 총합. 배경 실루엣(창문틀·나무 실루엣)은 개당 200–800 tri, DoF 블러 뒤에 위치.

### 6.4 아웃포커스 배경

- **내용**: 창문틀 어두운 수직선 1–2개 (극흐림), 창 너머 나무 실루엣 (색·밝기 덩어리만)
- **밝기**: 언제나 이끼 아이보다 명도 최소 25% 낮음
- **금지**: 읽힐 수 있는 텍스트, 얼굴, 의미 오브젝트

**기술 제약**: DoF는 허용. `Diaphragm Blades = 0`(원형 Bokeh), `Max Bokeh Size ≤ 4.0` (UX/TA 결정 T4).

### 6.5 21일 환경 변화 — 절제의 원칙

| 시기 | 허용 변화 |
|---|---|
| Day 1–7 | 병 표면 응결 약간 증가. 흙 표면 살짝 촉촉 |
| Day 8–14 | 소품 주변 먼지 미세 감소 (자주 본 흔적). 배경 빛 약간 낮아짐 |
| Day 15–21 | 병 주변 목재에 호박색 반사 면적 넓어짐. Farewell에서 먼지 파티클 미세 증가 |

**금지**: 계절 소품 교체, 배경 전면 교체, 새 오브젝트 등장. 변화는 **느끼지만 말할 수 없는 수준**.

### 6.6 무언의 스토리텔링

"누군가가 이 작은 세계를 돌본다"는 증거:
- 유리 표면의 구운 지문 흔적 1–2개
- 목재 표면의 동심원 수분 링 (오래 이 자리)
- 책상 연필 선 1–2개 (텍스처 레이어)
- 소품 책 1권이 특정 페이지에서 멈춤
- 마른 꽃이 병 옆에 기댄 형태

이 디테일들은 원칙 1의 "조심스럽게 들여다보는 사람"의 증거다. **그 사람이 플레이어이고, 그 시간이 게임이다.**

## 7. UI/HUD Visual Direction

### 7.1 다이어제틱 수준 — Semi-Diegetic 2D Composite

완전한 3D 다이어제틱은 드래그 구현이 과도하고, 완전한 스크린 스페이스는 원칙 3 위반. 절충:

- **카드**: 2D 스프라이트이되 씬 앞 레이어에 합성. 하단이 목재 표면과 닿은 것처럼 별도 그림자 적용
- **꿈 일기**: 화면 위로 올라오는 책 UI. 씬 앞 펼쳐지되 주변 호박색 빛 합성
- **타이틀/메뉴**: 병 옆 종이 쪽지

### 7.2 타이포그래피

**원칙**: 텍스트는 "쓰여진 것"이어야 한다. 디지털 스크린 텍스트가 아닌 종이에 인쇄되거나 손으로 쓴 것.

| 역할 | 폰트 | 용도 |
|---|---|---|

### 7.3 아이콘 언어

Section 3 "아이콘 기반 HUD 금지"를 준수. 아이콘의 정당한 이유는 **상태 변화 알림**만.

| 아이콘 | 스타일 | 근거 |
|---|---|---|
| 꿈 일기 알림 (닫힌 책 → 살짝 열린 책) | 선 드로잉, 단색, Nanum Myeongjo 필획 무게와 동일 | 색 + 형태 변화로 알림 (Section 4 백업) |
| 설정 버튼 | 종이를 접어 만든 탭 | Section 3 준수 |

**스타일 규칙**: 단선 스케치 — 동일 stroke width, 직각 없음, 둥근 모서리. 포토리얼·플랫 솔리드·그라디언트 금지. Nanum Myeongjo의 선 무게와 시각적 통일.

### 7.4 UI 애니메이션 (원칙 4 준수 — 이벤트만, 상시 루프 금지)

| 이벤트 | 연출 |
|---|---|
| 카드 등장 | 3장 순차(각 0.3초 간격). 아래에서 위로 소프트 슬라이드 + 페이드인. 최대 0.6초. **등장 후 정지** (상시 루프 금지) |
| **카드 hover (드롭존 근처)** | **이끼 아이가 카드 방향으로 미세 기울어짐(~5°) + SSS 증가 — UX 결정 U1. "이벤트에 대한 절제된 반응"으로 Stillness 준수** |
| 카드 건넴 확정 | 빛이 한 번 출렁이고 조용해짐. 카드 0.4초 페이드 아웃 + 미세 스케일 감소. **폭발·파티클 금지** |
| 꿈 텍스트 등장 | 글자 단위 left-to-right 페이드인(0.05초 지연/글자). **UMG Text 컴포넌트 + 마스크 알파 애니메이션 — UX 결정 U4. 렌더 텍스처 구현 금지 (스크린 리더 대응)** |
| 일기 열기/닫기 | 왼쪽 바인딩 페이지 오른쪽으로 열림, 1.0초 |

**금지**: 로딩 스피너, 상시 루프 글로우·펄스, 버튼 호버 스케일 팝업 (미세 색조 변화만 허용).

### 7.5 소형 창 (800×600) 적응

**우선순위**:
1. 이끼 아이 화면 높이 12% 이상 (원칙 1 — 타협 불가)
2. 카드 3장 동시 표시
3. 카드 제목 최소 14px

카드 최소 크기: **80×120px** (타로 비율). 조정 방식: 씬 비율 유지, UI 오버레이 위치만 변경. **소형 창에서 종이 질감 제거·sans-serif 교체 금지**.

### 7.6 상주 모드 (Always-on-Top) UI

- 카드 UI는 인터랙션 시작 전까지 **숨김**. 유리병 씬만
- 일기 아이콘 알림: 살짝 열린 책 형태만 (색 변화 없음 — 더 조용한 알림)
- 꿈 알림: Lavender Dusk 매우 약한 펄스 1회만
- **배경 블러 금지**: 게임은 그 자체로 존재하고 뒤를 가리지 않음

### 7.7 내비게이션 결정

| 영역 | 결정 |
|---|---|
| **"오늘이 며칠째인가" 표시** | **표시 안 함 (UX 결정 U3)**. 이끼 아이 성장 단계가 암묵적 시간 신호. 플레이테스트 후 재검토 가능 |
| **꿈 일기 탐색** | **선형 + 접힌 종이 귀퉁이 탭** (월/시즌 단위, UX 결정 U2). Section 3 "종이를 접어 만든 탭"과 정합 |
| **설정 진입** | Esc 키 또는 종이 탭 아이콘 → 비침습적 오버레이 |
| **복귀 플레이어 진입** | 타이틀 → 1-click → 즉시 오늘 세션. 저장 확인 팝업 없음. 자동 저장 |
| **Day 21 이후 UX** | `game-designer`와 협의 필요 (열린 결정) |
| **게임패드/Steam Deck 드래그** | `game-designer`와 협의 필요 — "건네는 제스처"가 물리적 은유로 필수인가, "의도 선택"이 핵심인가 |

### 7.8 접근성 요구사항 (신규 — UX 제기)

- **텍스트 스케일**: 3단계 선택 (기본/크게/매우 크게). 카드 크기 동반 확대 (비율 유지). art-director 재검토 필요
- **Reduced Motion**: 토글 — 브리딩 사이클 중단, 페이드인 즉시 컷, 파티클 제거, 책 플립 즉시 컷
- **스크린 리더**: 꿈 텍스트는 반드시 UMG Text + `AccessibleText` 필드 설정 (UX 결정 U4)
- **색각 이상**: Section 4 규정 준수. 주요 상태는 Deuteranopia/Protanopia 시뮬레이션 승인 필수

## 9. Reference Direction

각 레퍼런스에서 **정확히 무엇을 가져오고 무엇을 피하는지** 명시한다. "전반적인 느낌"은 금지 — 구체적 기법·구도·색·리듬만.

### 9.1 게임 레퍼런스

#### Reference A — **Cozy Grove** (Spry Fox, 2021)
- **가져오기**: 일일 세션 구조의 감정 리듬 — "오늘의 할 일은 적고, 그래서 매일 돌아온다"는 페이싱. 하루 15–20분짜리 세션의 감정 아크 구성
- **피하기**: 크래프팅/수집/스탯 UI. 주변 캐릭터의 수다스러움. "오늘도 많이 준비했어요!"식의 콘텐츠 밀도. 이 게임은 **훨씬 더 조용해야 한다**
- **중요한 이유**: 실시간 daily cozy가 유료($14.99)로 성립함을 증명. 단, Cozy Grove는 "섬 관리" 쪽으로 기울었고 Moss Baby는 **하나의 존재와의 관계**로 더 좁혀야 한다

#### Reference B — **Unpacking** (Witch Beam, 2021)
- **가져오기**: 대사 없는 서사. 오브젝트의 **배치/선택**만으로 캐릭터의 내면을 드러내는 UI 철학. 짧은 효과음과 정적 프레임의 결합 방식
- **피하기**: 방 전체 와이드 샷. 진척이 명시적으로 측정되는 감각(방 완료 게이지). Moss Baby는 **더 가까운 렌즈**를 유지한다
- **중요한 이유**: 내러티브 UI가 대사 UI를 이길 수 있음을 상업적으로 증명(200만+). 카드 아트의 "한 장으로 많은 것을 말한다" 철학의 근거

#### Reference C — **Strange Horticulture** (Bad Viking, 2022)
- **가져오기**: 종이·책·식물 표본의 질감 표현. Paper Warm 톤의 UI가 cozy 게임에서 어떻게 작동하는지. 화면 구도에서 **책상 위 오브젝트**의 레이아웃 레퍼런스
- **피하기**: 미스터리 해결 구조. 플레이어에게 주어지는 명시적 문제. Moss Baby는 **풀어야 할 문제가 없다**
- **중요한 이유**: 조용한 책상 top-down UI 게임이 성립하는 증거. 단, Strange Horticulture의 구도는 다소 문학적/건조함 — Moss Baby는 **더 부드럽고 따뜻**해야 한다

### 9.2 비게임 레퍼런스

#### Reference D — **Studio Ghibli (My Neighbor Totoro, Ponyo, Spirited Away의 조용한 장면들)**
- **가져오기**: 정령과의 조용한 만남의 순간 (토토로가 빗속에서 기다리는 씬, 하쿠가 처음 나타나는 씬). 정령을 **설명하지 않고 보여주는** 연출. 황혼 빛의 회화적 해석
- **피하기**: 대사가 있는 캐릭터 장면. 모험/플롯이 전개되는 장면. 화려한 색채의 액션 시퀀스. Moss Baby는 **Totoro가 말없이 앉아있는 30초**만을 지향한다
- **중요한 이유**: "정령 = 설명하지 않음"의 미적 권위. Ghibli 전체가 아닌 특정 *정지 장면*의 감성

#### Reference E — **Hobonichi Techo / 1cm 일기 / 작은 다이어리 문화 (일본·한국)**
- **가져오기**: 하루를 짧게 기록하는 의식의 미학. 격자 노트·작은 스티커·수수한 펜 색의 조합. 일기가 **디지털이 아닌 수공예 오브젝트**라는 감각
- **피하기**: 인스타그램식 "보여주기 위한" 정갈함. 완벽하게 배치된 플랫레이 사진. 스티커 쇼케이스. Moss Baby의 일기는 **플레이어 자신만을 위한 것**이다 (Anti-pillar: 소셜 비교 없음)
- **중요한 이유**: "짧은 일상 기록이 쌓여 의미가 되는" 구조가 대규모 소비문화로 존재함을 보여줌 (Hobonichi는 매년 수백만 권 판매). 꿈 일기 UI의 실체적 근거

### 9.3 레퍼런스 분포 도식

각 레퍼런스가 서로 다른 방향을 가리키는지 확인:

```
        (A) Cozy Grove
         세션 리듬 · daily cadence
                    │
(E) Hobonichi ─── Moss Baby ─── (B) Unpacking
일기/기록 문화              대사 없는 서사 / 오브젝트
                    │
         (C) Strange Horticulture            (D) Ghibli
         책상 UI · 종이 질감                   정령의 침묵
```

5개 레퍼런스가 서로 다른 5개 축(세션 리듬 / 서사 UI / UI 질감 / 정령 미학 / 기록 문화)을 지원한다. 어느 레퍼런스도 다른 레퍼런스와 같은 방향을 가리키지 않는다. 이것이 **"X의 카피"로 읽히지 않기 위한** 배치다.

### 9.4 단 하나의 금지 레퍼런스

- **Tamagotchi**의 **죽음·스트레스 페널티·배설물 관리**는 이 게임의 DNA가 아니다. 돌봄 구조는 가져오되 **불안과 의무는 배제**한다. 이 구분은 Pillar 1 (Quiet Presence)과 관련된 Anti-pillar의 본질이다.
